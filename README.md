# ⚡ 素因数分解バトル

素因数分解の速さと正確さを競う、2〜4人対戦の数学パーティーゲームです。  
Python + tkinter で動作するデスクトップアプリケーションです。

## ゲーム概要

各プレイヤーが「お題の数より大きい同じ桁の整数」を自分で選び、その数を素因数分解して、**素因数の個数（または種類数）が最も多い人**が勝ちとなるバトルゲームです。

- **プレイ人数**: 2〜4人（同じ端末でターン制）
- **形式**: 通常モードは BO3（先に2勝したプレイヤーが優勝）
- **高難易度モード**: 先に3勝するまでエンドレスで継続

---

## 動作環境・インストール

| 項目 | 要件 |
|------|------|
| Python | 3.10 以上（`list \| None` 型ヒント使用のため） |
| 外部ライブラリ | **なし**（標準ライブラリのみ） |
| GUI | tkinter（Python 標準付属） |
| OS | Windows / macOS / Linux |

> **macOS の場合**、システム付属の Python には tkinter が含まれていないことがあります。  
> `brew install python-tk` または [python.org](https://www.python.org/) からインストールした Python をお使いください。

## ゲームの流れ

```
① セットアップ画面
   ↓ プレイヤー名・桁数・難易度・制限時間を設定
② ラウンド開始（お題の数がランダムに表示される）
   ↓
③ 数の選択フェーズ（全員が順番に「お題より大きい同桁の整数」を入力）
   ↓
④ 素因数分解フェーズ（全員が順番に自分の選んだ数を素因数分解して入力）
   ↓
⑤ 結果フェーズ（正誤判定・スコア表示・ラウンド勝者の確定）
   ↓ 2勝（高難易度は3勝）先取で優勝へ
⑥ 優勝画面（最終スコア表示・リスタートまたは終了）
```

---

## 勝利条件

| モード | ラウンドの勝者 | ゲームの勝者 |
|--------|---------------|-------------|
| 通常モード | 正解した中で**素因数の個数**が最多のプレイヤー | **2勝先取** |
| 高難易度モード | 正解した中で**素因数の種類数**が最多のプレイヤー | **3勝先取** |

同点の場合、そのラウンドは両者敗北（どちらにも勝利は加算されません）。  
全員不正解の場合も同様です。

---

## 素因数の入力形式

以下の形式をすべて受け付けます。

| 入力例 | 意味 |
|--------|------|
| `2 2 3` | スペース区切り |
| `2^2 3` | べき乗表記（2² × 3） |
| `2*2*3` | アスタリスク区切り |
| `2×2×3` | 全角「×」区切り |
| `2x2x3` | 小文字「x」区切り |
| `2^2*3` | べき乗とアスタリスクの混在 |

---

## 設定オプション

### プレイヤー数
2〜4人まで設定可能。デフォルトの名前は Alice / Bob / Carol / Dave。

### お題の桁数
2桁 / 3桁 / 4桁 から選択。桁数が大きいほど難しくなります。

### 難易度
- **通常**: 素因数の個数（重複あり）で競う。例：`12 = 2 × 2 × 3` → スコア 3
- **高難易度**: 素因数の種類数（重複なし）で競う。例：`12 = 2 × 2 × 3` → スコア 2

### 制限時間
チェックを入れると素因数分解フェーズに制限時間（10〜120秒）が設定されます。
- 残り10秒で黄色、残り5秒以下で赤に変わります。
- 時間切れは「未回答（空の解答）」として扱われ不正解になります。

## 主要関数の説明

| 関数名 | 説明 |
|--------|------|
| `prime_factorize(n)` | 整数 `n` を素因数分解し、素因数のリストを返す |
| `is_correct(n, factors)` | `factors` が `n` の正しい素因数分解かどうかを検証する |
| `parse_factors(s)` | 入力文字列をパースして素因数のリストに変換する |
| `generate_topic(digits)` | 指定した桁数のランダムな整数を生成してお題にする |
| `digit_count(n)` | 整数 `n` の桁数を返す |
